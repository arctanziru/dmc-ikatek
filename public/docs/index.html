Introduction

Base URL: http://dmc-ikatek.test

This documentation aims to provide all the information you need to work with our API.

As you scroll, you'll see code examples for working with the API in different programming languages in the dark area to the right (or as part of the content on mobile). You can switch the language used with the tabs at the top right (or from the nav menu at the top left on mobile).

Authenticating requests

This API is not authenticated.

Authentication

APIs for managing authentication

Login User

Authenticate the user and return an access token.

Example request:
curl --request POST \
    "http://dmc-ikatek.test/api/v1/auth/login" \
    --header "Content-Type: application/json" \
    --header "Accept: application/json" \
    --data "{
    \"email\": \"john@example.com\",
    \"password\": \"password123\"
}"

const url = new URL(
    "http://dmc-ikatek.test/api/v1/auth/login"
);

const headers = {
    "Content-Type": "application/json",
    "Accept": "application/json",
};

let body = {
    "email": "john@example.com",
    "password": "password123"
};

fetch(url, {
    method: "POST",
    headers,
    body: JSON.stringify(body),
}).then(response => response.json());

Example response (200):


{
    "status": "success",
    "message": "Login success",
    "data": {
        "user": {
            "id": 1,
            "name": "John Doe",
            "email": "john@example.com",
            "username": "johndoe",
            "role": "user"
        },
        "token": "your_generated_token_here"
    }
}

Example response (401):


{
    "status": "error",
    "message": "Invalid credentials"
}

Logout User

requires authentication

Revoke the user's access token and log them out.

Example request:
curl --request POST \
    "http://dmc-ikatek.test/api/v1/auth/logout" \
    --header "Authorization: Bearer your_generated_token_here" \
    --header "Content-Type: application/json" \
    --header "Accept: application/json"
const url = new URL(

Get a list of news categories.

Example request:
curl --request GET \
    --get "http://dmc-ikatek.test/api/v1/news-category?search=technology&page=1&per_page=10" \
    --header "Content-Type: application/json" \
    --header "Accept: application/json"
const url = new URL(
    "http://dmc-ikatek.test/api/v1/news-category"
);

const params = {
    "search": "technology",
    "page": "1",
    "per_page": "10",
};
Object.keys(params)
    .forEach(key => url.searchParams.append(key, params[key]));

const headers = {

    method: "GET",
